Вариант базы данных: Кинотеатр

Скрипты создания таблиц запускать по порядку от 1 до 5. В таком порядке нормально создаются ссылки от вторичного ключа к первичному.


Ну и сами запросы 6-9 файлы:

	Select 1: Вывод всех фильмов показываемых в конкретную дату. в данном случае на 9 мая.

	Select 2: Вывод ближайшего сеанса, одного фильма.

	Select 3: Вывод, сколько человек в сумме сходило на каждый фильм за весь период работы кинотеатра.

	Select 4: Вывод суммы стоимости всех билетов за месяц. У меня выведено за май.
